Para que funcione es necesario tener instalado un docker
despues se proccede a la termial de Linux o a powershell en caso de Windows
Y procedemos a la carpeda del proyecto con cd y la direccion
en este vamos a usar el comando 
sudo docker-compose up --build (linux)
docker-compose up --build (windows)
esto para levantar en doker en tu sistema operativo por mprimer vez
ya en otras ocaciones basta con usa
sudo docker-compose up (linux)
docker-compose up (windows)
ya para finalizar se usa
docker ps
para checar que este funcionando
y
docker-compose down
para detener el programa
